# **App Name**: Droop ERP

## Core Features:

- Multi-Tenant Authentication: Firebase Authentication setup with tenant-aware access control for different companies.
- Subscription Management: Implement subscription tiers (Free, Standard, Custom) with Firebase Functions to manage access.
- Point of Sale (POS): Develop a POS module with Shop and Restaurant modes for different sales environments.
- Modular Dashboard: Design a dashboard with a modular layout similar to Odoo, using icons for easy navigation.
- DATEV CSV Export: Enable export of financial data in DATEV CSV format using Firebase Functions.
- GDPR Compliance Tools: Implement tools for data export and deletion, including consent logging and audit trails for GDPR compliance, which also uses an AI tool to discover related data.
- Invoice & Asset Storage: Utilize Firebase Storage to securely store invoices and assets with appropriate access controls.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke trust and reliability.
- Background color: Light grayish-blue (#ECEFF1) for a clean, professional look.
- Accent color: Vibrant orange (#FF9800) for interactive elements and calls to action.
- Headline font: 'Space Grotesk', sans-serif, for headlines. Body font: 'Inter', sans-serif, for body text.
- Use flat, modern icons with a consistent style across the dashboard.
- Design a modular, grid-based layout for the dashboard to ensure responsiveness.
- Subtle animations to enhance user experience during navigation and data loading.
